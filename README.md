**Description:**  
**EasyModes** adds full-featured **BUILD** and **PVP** game modes to Garry's Mod with extended features and a clean, intuitive interface. Perfect for servers focused on building and PvP.

## Features

- **Two game modes:** BUILD and PVP.
- **ULX Support:** admins can manage players and modes through ULX commands.
- **2D and 3D HUD:** informative interface for all players.
- **Addon-friendly:** designed to minimize conflicts with other mods.

## Installation

1. Download.
2. Place the **EasyModes** folder in `garrysmod/addons/`.
3. Restart your server or client.
4. Make sure the ULX is installed.

## HUD

- **2D HUD:** display current mode.
- **3D HUD:** displays other players current modes.
